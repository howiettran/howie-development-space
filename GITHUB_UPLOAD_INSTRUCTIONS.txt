HOW TO BUILD HOWIE TRANSLATE v0.10.1

1. Extract this ZIP.
2. Open the local howie-development-space repository folder.
3. Delete the old project files, but keep the hidden .git folder.
4. Copy everything inside this extracted howie_translate_src folder into the repository folder.
5. Open GitHub Desktop.
6. Enter summary: Build Howie Translate v0.10.1 translation startup fixes
7. Click Commit to main.
8. Click Push origin.
9. Open the repository on GitHub and select Actions.
10. Open the completed Build Howie Translate APK run.
11. Under Artifacts, download Howie-Translate-v0.10.1-test-APK.
12. Unzip that artifact and install Howie-Translate-v0.10.1-test.apk.

TEST FIRST
- Translate a short English phrase into Chinese, then translate the result back into English.
- Start Live with English and Chinese selected and confirm it leaves Preparing mode and begins listening.
- In Settings, run Download all translation models and confirm it processes Chinese, Vietnamese, Thai and Malay without an English-model error.
- Record a short Conversation, stop it, and confirm Open works in History and Saved.
- Export MP3 from History and Saved.
- Export a short MP4 from History and Saved.
- Capture a Chinese image, translate it, then confirm the original image appears in History and Saved.
- Confirm the selected MP3, MP4 and Image folders are used.
