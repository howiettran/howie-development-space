HOW TO UPLOAD AND BUILD THE APK

1. Open your cloned howie-development-space folder on the computer.
2. Extract this ZIP and open the howie_translate_src folder.
3. Copy everything inside howie_translate_src into the cloned repository folder, replacing older files when asked.
4. Return to GitHub Desktop.
5. Enter summary: Update Howie Translate to v0.9.1
6. Click Commit to main, then Push origin.
7. Open the repository in your browser and click Actions.
8. Open Build Howie Translate APK and wait for a green tick.
9. Open the completed run. Under Artifacts, click Howie-Translate-v0.9.1-test-APK.
10. Unzip the downloaded artifact and install Howie-Translate-v0.9.1-test.apk on the Android phone.

This test APK uses GitHub's debug signing key. Android may require removal of an older build that was signed with a different key. Back up anything important before uninstalling an older beta.
